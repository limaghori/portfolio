# React Developer Portfolio

This is a modern, responsive, and dynamic portfolio built with React and Vite. It features an interactive projects grid with category filtering and a fully functional dynamic dark mode.

## 🌐 Live Deployment
You can view the live website here: 
https://comforting-daffodil-38d0a3.netlify.app/

---

## 📝 Brief Reflection

Through this project, I mastered the skill of managing global themes in React using hooks like useState and syncing them with the DOM through useEffect. 

The biggest challenge I faced was overriding core styles to ensure the dark mode toggles perfectly across all project sections without layout breaking, which required extensive CSS debugging. 

My favorite creative feature is the interactive filter on the projects grid alongside the smooth background transition, as it significantly enhances the user experience and provides a modern feel to the portfolio.
